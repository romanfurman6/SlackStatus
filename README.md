# SlackStatus

*Mac OS app for changing "Slack" status.*

## Support

min: macOS 10.14 (Mojave)

*because of an issue with default emoji picker in previous macOS versions*

## Install
1) Double click on SlackStatus
2) Sign-in in workspace
3) Ready to use
